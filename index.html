Hometown Parser

Source code and more information: https://github.com/rendall/hometown-parser-2

This project extracts town and city data from natural-language paragraphs. Enter a paragraph below that mentions cities, towns, provinces, states or countries

Currently it is expansive, in that it will flag all possible locations with the same name (e.g. entering San Francisco will show all 6 cities with that name around the world). Having an exclusive search could be done without too much modification (e.g. entering San Francisco, Philippines would flag only that city
